EyeFiNote

An open source Eye-Fi Server which automatically loads files into Evernote

Download this project as a .zip file Download this project as a tar.gz file

Eye-Fi to Evernote

Now that Eyefi has discontinued support for automatically sending videos and images to Evernote without a subscription you can run this and when your Eyefi card is connected to the same network as the server running this project the images and videos will be automatically downloaded from the card and uploaded to Evernotte.

CAUTION There are effectively no safeguards in this code. If something goes wrong the data could well be lost forever.

Attributions

I've rolled a simple modification EyeFiServer by @tachang which will write the images into Evernote (currently relying on Geeknote's oAuth implementation to get an authorizattion token) rather than simply dumping them to the filesystem.
